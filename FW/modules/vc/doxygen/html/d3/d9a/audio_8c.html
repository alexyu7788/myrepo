The audioplay_create() function creates the audioplay object.

Parameters
handleOn success, this is filled in with a handle to use in other API functions.
sample_rateThe sample rate, in samples per second, for the PCM data. This shall be between 8000 and 96000.
num_channelsThe number of channels for the PCM data. Must be 1, 2, 4, or 8. Channels must be sent interleaved.
bit_depthThe bitdepth per channel per sample. Must be 16 or 32.
num_buffersThe number of buffers that will be created to write PCM samples into.
buffer_sizeThe size in bytes of each buffer that will be used to write PCM samples into. Note that small buffers of less than a few Kb in size may be faster than larger buffers, although this is platform dependent.
Returns
0 on success, -1 on failure.
