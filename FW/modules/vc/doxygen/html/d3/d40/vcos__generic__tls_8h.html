Detailed Description

Do an emulation of Thread Local Storage. The platform needs to provide a way to set and get a per-thread pointer which is where the TLS data itself is stored.

Each thread that wants to join in this scheme needs to call vcos_tls_thread_register().

The platform needs to support the macros/functions _vcos_tls_thread_ptr_set() and _vcos_tls_thread_ptr_get().
