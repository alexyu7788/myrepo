Detailed Description

This structure represents a tunnel in the OpenMAX IL API.

Some operations in this API act on a tunnel, so the tunnel state structure (TUNNEL_T) is a convenient store of the source and sink of the tunnel. For each, a pointer to the relevant component state structure and the port index is stored.
