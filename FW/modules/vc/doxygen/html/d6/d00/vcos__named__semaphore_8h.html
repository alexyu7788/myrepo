◆ vcos_named_semaphore_wait()

VCOS_INLINE_DECL void vcos_named_semaphore_wait ( VCOS_NAMED_SEMAPHORE_T *  sem)

Wait on a named semaphore.

There is no timeout option on a semaphore, as adding this will slow down implementations on some platforms. If you need that kind of behaviour, use an event group.

This always returns VCOS_SUCCESS and so should really be a void function. However too many lines of code would need to be changed in non-trivial ways, so for now it is non-void.

Parameters
semSemaphore to wait on
Returns
VCOS_SUCCESS - semaphore was taken.
