Detailed Description

Simple reply protocol. The client creates a semaphore and waits for it. No queuing of multiple replies is possible but nothing needs to be setup in advance. Because creating semaphores is very fast on VideoCore there's no need to do anything elaborate to optimize create time - this might need revisiting on other platforms.
