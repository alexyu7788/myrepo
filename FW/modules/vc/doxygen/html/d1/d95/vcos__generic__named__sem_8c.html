Detailed Description

Named semaphores, primarily for VCFW.

Does not actually work across processes; merely emulate the API.

The client initialises a VCOS_NAMED_SEMAPHORE_T, but this merely points at the real underlying VCOS_NAMED_SEMAPHORE_IMPL_T.

typedef struct VCOS_NAMED_SEMAPHORE_IMPL_T VCOS_NAMED_SEMAPHORE_IMPL_T

Each actual real semaphore is stored in one of these. Clients just get a structure with a pointer to this in it.

It also contains a doubly linked list tracking the semaphores in-use.
