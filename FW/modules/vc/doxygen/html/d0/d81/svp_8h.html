Detailed Description

Simple video player using MMAL. Uses MMAL container reader plus video decode component, and provides callback to retrieve buffers of decoded video frames.

Thread-safety: The public API functions must be called from the same thread for a given SVP_T instance. The user is notified of decoded frames and other events from a separate thread, started by svp_start().
