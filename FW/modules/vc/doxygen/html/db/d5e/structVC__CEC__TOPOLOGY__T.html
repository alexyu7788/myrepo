Detailed Description

CEC topology struct

Meaning of device_attr is as follows (one per active logical device) bit 3-0 logical address (see CEC_AllDevices_T above) bit 7-4 device type (see CEC_DEVICE_TYPE_T above) bit 11-8 index to upstream device bit 15-12 number of downstream device bit 31-16 index of first 4 downstream devices

To keep life simple we only show the first 4 connected downstream devices
